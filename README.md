# 🏋️ Gym Workout Data Analysis (Python)

This project analyzes gym workout data using Python, Pandas, NumPy, and Matplotlib.

The goal was to practice real-world data analysis — loading data, cleaning it, performing statistical calculations, and creating visualizations.

## What I did

- Loaded and cleaned the dataset  
- Checked for missing values and duplicates  
- Calculated mean, min, max, and standard deviation  
- Analyzed correlation between workout duration and calories burned  
- Grouped data by workout type and fitness level  
- Created multiple visualizations using Matplotlib  

## Visualizations

- Scatter plot (Duration vs Calories Burned)  
- Bar chart (Average Calories by Workout Type)  
- Pie chart (Workout Distribution)  
- Histogram (Age Distribution)  
- Combined dashboard  

## What I learned

This project helped me better understand data cleaning, numerical analysis with NumPy, and data visualization using Matplotlib.
